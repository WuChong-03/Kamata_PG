## 什么是标准输入输出

标准输入输出是程序最基础的交互方式之一。  

- 输出：把内容显示到控制台
- 输入：从键盘接收用户输入

在 C 语言中，标准输入输出通常使用 `stdio.h` 头文件中的函数完成。

---

## printf函数

`printf` 用于向控制台输出内容。

基本示例：

```c
#include <stdio.h>

int main() {
    printf("Hello");
    return 0;
}
```

输出结果：

```text
Hello
```

`printf` 也可以输出变量或多个内容，后续会继续学习格式化输出。

---

## scanf函数

`scanf` 用于从键盘接收输入，并把输入结果保存到变量中。

基本示例：

```c
#include <stdio.h>

int main() {
    int num = 0;
    scanf("%d", &num);
    return 0;
}
```

这里的 `%d` 表示读取整数，`&num` 表示把输入结果存入变量 `num` 中。

---

## 换行符

在输出时，常用 `\n` 表示换行。

示例：

```c
#include <stdio.h>

int main() {
    printf("Hello\n");
    printf("World\n");
    return 0;
}
```

输出结果：

```text
Hello
World
```

换行符的作用是让后续输出从下一行开始显示。

---

## 总结

标准输入输出函数是学习 C 语言时最早接触的一组函数。  

- `printf` 用于输出
- `scanf` 用于输入
- `\n` 用于换行

掌握这些内容后，程序就可以开始进行最基础的人机交互。
