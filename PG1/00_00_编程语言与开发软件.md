## 编程语言

### 市面上主要使用的语言

常见编程语言及其主要应用方向：

- C / C++ → 高性能程序、游戏开发、系统开发  
- Python → 数据分析、人工智能、自动化  
- Java → 企业级服务器后端开发  
- JavaScript → Web前端开发  
- C# → Unity游戏开发、桌面应用  


不同语言在性能、开发效率和使用场景上各有侧重。  

实际上，编程语言并不是严格限制用途的，例如《Minecraft》最早使用 Java 开发，说明 Java 也可以用于游戏开发。  
而后来发布并登陆 Windows 商店的基岩版则使用 C++ 开发，其性能相比早期 Java 版本更强。  

因此，不同语言在性能和开发效率等方面存在差异，在实际开发中通常会根据需求选择更合适的语言。

---

### 我们主要使用的 C 和 C++

#### C语言

特点：
- 面向过程，执行顺序清晰  
- 可以直接操作内存  
- 性能高  

适合：
- 系统开发  
- 嵌入式开发  
- 底层程序  

---

#### C++

特点：
- 在C基础上扩展  
- 支持面向对象（类、继承、多态）  
- 支持泛型（模板）  
- 保持高性能  

适合：
- 游戏开发  
- 游戏引擎  
- 高性能应用  

---

#### 为什么选择 C / C++（游戏开发）

游戏开发对性能要求极高，例如：

- 实时计算（位置、碰撞）  
- 图形渲染  
- 输入响应  

C / C++ 的优势：
- 执行速度快  
- 控制力强  
- 被主流游戏引擎广泛使用（如 Unreal Engine）

## 开发环境

### 市面上常见开发环境

不同语言通常配合不同开发工具：

- C / C++ → Visual Studio、CLion  
- Java → IntelliJ IDEA  
- Python → VSCode、PyCharm  
- Web开发 → VSCode  
- Unity（C#）→ Visual Studio  

---

### 我们使用的 Visual Studio

主要功能：

- **编写代码**  : 类似 VSCode，就是“写代码”
- **创建项目**  : 创建一个带结构的工程文件夹(解决方案+项目)
- **编译程序**  : 编译是“看不见的过程”,由软件内部自动完成  
我们可以设置编码（UTF-8）, 编译选项, 优化等级等等
- **运行程序(RUN)**    : 把 .exe 程序跑起来,你只能看结果
- **调试程序(DeBug)**  : 程序可以中断,可以断点调试

---

### 开发环境与语言的关系

编程语言和开发环境是不同的概念：

- 编程语言 → 用来描述程序逻辑  
- 开发环境 → 用来编写和运行代码  

它们之间的关系是“配合使用”：

- 一种语言可以使用多种开发环境  
- 一个开发环境也可以支持多种语言  

例如：
- C++ 可以用 Visual Studio、CLion、VSCode  
- Python 可以用 VSCode、PyCharm  

---

## 总结

- 编程语言决定程序的实现方式  
- 开发环境决定程序的开发方式  
- C / C++ 由于高性能，被广泛用于游戏开发  
- Visual Studio 是学习 C / C++ 的主要工具
