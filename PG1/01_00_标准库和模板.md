## 什么是标准库

标准库是 C 语言提供的一组常用函数集合。  
这些函数已经由系统或编译环境准备好，可以直接使用。

标准库主要用于完成常见任务，例如：

- 输入输出
- 数学计算
- 字符串处理
- 内存管理
- 时间处理

使用标准库函数前，通常需要先包含对应的头文件。

---

## 常见标准库

| 头文件 | 主要用途 | 常用函数 |
|--------|----------|----------|
| `stdio.h` | 输入输出 | `printf`, `scanf`, `puts` |
| `stdlib.h` | 通用功能 | `malloc`, `free`, `atoi`, `exit` |
| `math.h` | 数学计算 | `sqrt`, `pow` |
| `string.h` | 字符串处理 | `strlen`, `strcpy`, `strcmp` |
| `time.h` | 时间处理 | `time`, `clock` |
| `ctype.h` | 字符判断 | `isalpha`, `isdigit`, `toupper` |

---

## 什么是程序模板

在学习 C 语言时，程序通常会从固定的基础结构开始：

```c
#include <stdio.h>

int main() {
    return 0;
}
```

这类固定结构可以看作程序模板。  

---

## 程序模板的作用

### `#include <stdio.h>`
用于引入标准输入输出库，使程序可以使用 `printf` 等函数。

### `int main()`
表示程序的入口函数，程序会从这里开始执行。

### `return 0;`
表示程序正常结束，并把结果返回给系统。

程序模板的意义在于：  
它提供了一个最基础、最标准的程序起点，便于后续继续添加代码。

---

## 总结

标准库提供常用功能，程序模板提供基础结构。  

学习 C 语言时，通常会先从模板开始，再逐步加入标准库函数和程序逻辑。
